一个有状态的组件

					

						除了输入数据(通过访问 this.props),一个组件可以保持内部状态数据(通过访问 this.state)。当组件的状态数据发生变化时,所呈现的标记将被重新调用更新render().
					

					
Live JSX Editor
Compiled JS
xxxxxxxxxx
 
​
var Timer = React.createClass({
  getInitialState: function() {
    return {secondsElapsed: 0};
  },
  tick: function() {
    this.setState({secondsElapsed: this.state.secondsElapsed + 1});
  },
  componentDidMount: function() {
    this.interval = setInterval(this.tick, 1000);
  },
  componentWillUnmount: function() {
    clearInterval(this.interval);
  },
  render: function() {
    return (
      <div>Seconds Elapsed: {this.state.secondsElapsed}</div>
    );
  }
});
​
ReactDOM.render(<Timer />, mountNode);
​
Seconds Elapsed: 259

				

				

					
一个应用程序

					

						使用 props 和 state,我们可以把一个小Todo应用程序。这个示例使用  state跟踪当前项目列表以及用户输入的文本。虽然事件处理程序似乎是呈现内联,它们也会通过事件代理来收集和实现。

var TodoList = React.createClass({
  render: function() {
    var createItem = function(item) {
      return <li key={item.id}>{item.text}</li>;
    };
    return <ul>{this.props.items.map(createItem)}</ul>;
  }
});
var TodoApp = React.createClass({
  getInitialState: function() {
    return {items: [], text: ''};
  },
  onChange: function(e) {
    this.setState({text: e.target.value});
  },
  handleSubmit: function(e) {
    e.preventDefault();
    var nextItems = this.state.items.concat([{text: this.state.text, id: Date.now()}]);
    var nextText = '';
    this.setState({items: nextItems, text: nextText});
  },
  render: function() {
    return (
      <div>
        <h3>TODO</h3>
        <TodoList items={this.state.items} />
        <form onSubmit={this.handleSubmit}>
          <input onChange={this.onChange} value={this.state.text} />
          <button>{'Add #' + (this.state.items.length + 1)}</button>
        </form>
      </div>
    );
  }
});
​
ReactDOM.render(<TodoApp />, mountNode);

						React是挺灵活的,它提供钩子,让你与其他库和框架接口。这个示例使用remarkable,外部Markdown库,将实时文本区域的价值。

    var MarkdownEditor = React.createClass({
      getInitialState: function() {
        return {value: 'Type some *markdown* here!'};
      },
      handleChange: function() {
        this.setState({value: this.refs.textarea.value});
      },
      rawMarkup: function() {
        var md = new Remarkable();
        return { __html: md.render(this.state.value) };
      },
      render: function() {
        return (
          <div className="MarkdownEditor">
            <h3>Input</h3>
            <textarea
              onChange={this.handleChange}
              ref="textarea"
              defaultValue={this.state.value} />
            <h3>Output</h3>
            <div
              className="content"
              dangerouslySetInnerHTML={this.rawMarkup()}
            />
          </div>
        );
      }
    });
    ​
    ReactDOM.render(<MarkdownEditor />, mountNode);
